## Reporting a Vulnerability

We take the security of `GitHub Copilot API Gateway` seriously. If you have found a security vulnerability in this project, please report it to us immediately. 

**Do NOT report security vulnerabilities on the public GitHub issue tracker.**

To report a vulnerability, please email **suhaib.bin.younis@gmail.com**. Assuming it is valid, we will work with you to fix the vulnerability and give you credit for the discovery.
